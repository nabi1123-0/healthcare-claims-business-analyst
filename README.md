## Project Overview
This project simulates a real-world healthcare insurance scenario where claims data is analyzed to support business and operational decision-making. The goal is to identify cost drivers, utilization trends, and pricing insights that would help health plans improve efficiency and control costs.

This project reflects responsibilities typically handled by a Business Analyst or Data Analyst in healthcare environments such as health plans and insurance providers.

---

## Business Objectives
- Analyze healthcare claims data to identify high-cost procedures
- Compare claim costs across plan types
- Understand utilization trends
- Provide data-driven insights to support pricing and operational decisions

---

## Dataset
The dataset represents simulated healthcare insurance claims and includes:
- Member ID
- Plan Type (HMO, PPO, EPO)
- Claim Amount
- Procedure Category
- In-Network / Out-of-Network indicator
- Claim Date

---

## Tools & Technologies
- SQL (MySQL-style queries)
- Python (Pandas, NumPy)
- Excel (optional reporting)
- GitHub for version control

---

## Key Analysis Performed
- Total and average claim cost analysis
- Cost comparison by plan type
- Identification of high-cost procedures
- In-network vs out-of-network cost analysis
- Trend analysis over time

---

## Business Impact
The insights from this analysis can help healthcare organizations:
- Optimize pricing strategies
- Reduce unnecessary high-cost claims
- Improve network utilization
- Support strategic decision-making
